# [Book App](https://khairul-abdi.github.io/portfolio/ "Live Demo")

### This app uses :

- Node.js
- Express.js
- Ejs.js
- Sqlite3
- Bootstrap

## Quick Start

``` bash
# Install dependencies
npm install

# Serve on localhost:5000
npm start
# Dev Server (Nodemon)
npm run dev
```
